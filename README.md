# Project `EcomGen`

- 개요 : 이커머스 셀러들을 위한 `상품 설명 생성모델`, 더 이상 상품 설명을 위해 비싼 비용 들이지 않기 위해 만든 모델 


- 주요 현황 : 약 5개의 모델을 통해 성능 비교 

> 최종 선정 모델 
> 1. `Polyglot-1.3b`
> 2. `Gemma2-ko-2b-it`
